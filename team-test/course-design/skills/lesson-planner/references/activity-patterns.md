# Activity patterns library

Reusable instructional activity templates organized by Bloom's level and delivery format.

## Remember level activities

### Flashcard review
- **Format:** individual, any delivery
- **Time:** 5-10 min
- **How:** learners review key terms / definitions using flashcards (physical or digital like Anki)
- **Good for:** vocabulary, terminology, tool names and purposes

### Labeling exercise
- **Format:** individual, any delivery
- **Time:** 5-10 min
- **How:** given a diagram / screenshot, learners label components
- **Good for:** interface familiarity, system architecture, process steps

## Understand level activities

### Concept mapping
- **Format:** individual or pairs, any delivery
- **Time:** 15-20 min
- **How:** create a visual map connecting related concepts with labeled relationships
- **Good for:** showing how ideas relate, revealing misconceptions

### Jigsaw
- **Format:** groups (4-6), synchronous
- **Time:** 30-45 min
- **How:** each person becomes "expert" on one subtopic, then teaches their group
- **Good for:** large bodies of content, building teaching skills

### Analogy challenge
- **Format:** pairs or small groups, any delivery
- **Time:** 10-15 min
- **How:** "Explain [concept] to someone who's never heard of it, using an analogy from everyday life"
- **Good for:** deepening understanding, revealing gaps

## Apply level activities

### Guided walkthrough
- **Format:** individual, any delivery
- **Time:** 15-30 min
- **How:** step-by-step instructions with scaffolding. Learner follows along, making small decisions at each step.
- **Good for:** first exposure to a tool or process

### Scenario-based exercise
- **Format:** individual or pairs, any delivery
- **Time:** 20-40 min
- **How:** "You're [role] and [situation]. Using [tool / method], do [task]."
- **Good for:** contextualizing skills in realistic situations

### Code-along / build-along
- **Format:** individual with instructor, synchronous
- **Time:** 20-45 min
- **How:** instructor demonstrates, learners follow on their own machines. Pause points for learners to try variations.
- **Good for:** technical skills, tool proficiency

### Drill set
- **Format:** individual, any delivery
- **Time:** 15-30 min
- **How:** multiple short exercises of the same type with increasing difficulty
- **Good for:** building fluency and speed

## Analyze level activities

### Case study analysis
- **Format:** small groups (3-5), any delivery
- **Time:** 30-60 min
- **How:** given a real-world case with data, identify patterns, root causes, or key factors. Present findings.
- **Good for:** complex problem decomposition, pattern recognition

### Compare and contrast
- **Format:** pairs, any delivery
- **Time:** 20-30 min
- **How:** given two approaches / tools / solutions, systematically compare across defined criteria
- **Good for:** developing judgment, understanding trade-offs

### Debugging / troubleshooting challenge
- **Format:** individual or pairs, any delivery
- **Time:** 20-40 min
- **How:** given a broken system / process / output, diagnose what's wrong and explain why
- **Good for:** analytical thinking, system understanding

### Data interpretation
- **Format:** individual or small groups, any delivery
- **Time:** 20-30 min
- **How:** given a dataset, chart, or report, answer specific analytical questions
- **Good for:** data literacy, evidence-based reasoning

## Evaluate level activities

### Peer review
- **Format:** pairs, any delivery
- **Time:** 20-30 min
- **How:** review a peer's work using a provided rubric. Give specific, constructive feedback.
- **Good for:** developing critical assessment skills, seeing different approaches

### Tool / solution evaluation
- **Format:** individual or small groups, any delivery
- **Time:** 30-60 min
- **How:** evaluate options against defined criteria, weigh trade-offs, make and defend a recommendation
- **Good for:** decision-making skills, professional judgment

### Retrospective / post-mortem
- **Format:** small groups or full group, synchronous
- **Time:** 20-30 min
- **How:** after completing a project or exercise: what worked? What didn't? What would you change?
- **Good for:** reflective practice, continuous improvement mindset

### Gallery walk with critique
- **Format:** full group, synchronous
- **Time:** 30-45 min
- **How:** each person / group posts their work. Everyone walks around, leaves feedback on sticky notes or comments.
- **Good for:** exposure to multiple approaches, building evaluation skills

## Create level activities

### Mini-project
- **Format:** individual or pairs, any delivery
- **Time:** 1-3 hours
- **How:** build something that integrates skills from the current and previous modules. Open-ended but with clear deliverables.
- **Good for:** synthesis, creative application

### Design challenge
- **Format:** small groups, synchronous preferred
- **Time:** 45-90 min
- **How:** given a problem and constraints, design a solution. Present and defend your design.
- **Good for:** creative problem-solving, dealing with ambiguity

### Capstone project
- **Format:** individual or teams, extended timeline
- **Time:** multiple sessions / 1-2 weeks
- **How:** a substantial project that integrates all course skills. Includes planning, execution, and presentation.
- **Good for:** final assessment, portfolio building, real-world application

### Teach-back
- **Format:** individual or pairs, synchronous
- **Time:** 15-30 min per person
- **How:** learner teaches a concept or skill to peers. Must create their own materials.
- **Good for:** deepest level of understanding, communication skills

## Delivery format adaptations

### Making synchronous activities work async

| Sync activity | Async adaptation |
|---------------|------------------|
| Group discussion | Discussion forum with required replies |
| Live demo | Recorded walkthrough + follow-along exercise |
| Peer review | Async review with rubric, 48-hour turnaround |
| Gallery walk | Shared gallery board (Miro, Padlet) with comment period |
| Think-pair-share | Reflection prompt → forum post → reply to one peer |

### Making individual activities work in groups

| Individual activity | Group adaptation |
|---------------------|------------------|
| Drill set | Relay race (team members take turns, fastest team wins) |
| Scenario exercise | Same scenario, different roles, combine outputs |
| Case study analysis | Divide case into sections, jigsaw back together |
| Mini-project | Larger scope, divided into clear individual deliverables |
